# Expense Tracker

A single-page personal expense tracker built with plain HTML, CSS, and
JavaScript for CSCI 39548 Assignment 2

## Features

- Add an expense with a description, amount, category, and date (the date
  defaults to today)
- See all expenses in a table, with the total amount, count, and a
  breakdown by category
- Filter the list to show only one category, or sort it by date or amount
- Delete any expense from the list
- If you enter something invalid, you'll see an error message and the expense won't be added
- If there are no expenses to show, you'll see a friendly empty message
  instead of a blank table
- Your expenses are saved automatically, so they're still there if you
  refresh the page
- Click "Convert to EUR" to see your overall total converted to euros using
  live exchange rates

## Structure

- index.html — the page structure
- styles.css — the page's styling and layout
- script.js — the app's logic
